Política de Privacidade

Última atualização: 30 de maio de 2026

O Lapzy é um aplicativo de cronometragem de voltas para kart desenvolvido por Ricardo Silva. Esta política descreve quais dados são coletados durante o uso do app, como são utilizados e como são protegidos.

Dados coletados

O Lapzy coleta apenas os dados estritamente necessários para o funcionamento da cronometragem:

  • Localização precisa (GPS) — coletada em primeiro plano durante uma sessão de corrida ativa. Usada exclusivamente para detectar cruzamentos da linha de largada/chegada e fronteiras de setor. Não é armazenada de forma contínua nem enviada a servidores externos.
  • Nome de dispositivos Bluetooth — exibido na tela de configuração de GPS externo quando o usuário opta por usar um receptor GPS via Bluetooth. Não é armazenado permanentemente nem associado a identificadores pessoais.
  • Dados de corrida — tempos de volta, tempos por setor, data e pista. Salvos localmente no dispositivo do usuário. Não são enviados a nenhum servidor sem ação explícita do usuário.

O que não coletamos

  • Nome, e-mail ou qualquer dado de conta
  • Identificadores únicos de dispositivo (IMEI, número de série)
  • Localização em background (fora de uma corrida ativa)
  • Dados de saúde ou biométricos
  • Histórico de navegação ou uso de outros apps
  • Dados financeiros ou de pagamento

Permissões do sistema

  • ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION — necessárias para o GPS cronometrar voltas.
  • FOREGROUND_SERVICE / FOREGROUND_SERVICE_LOCATION — mantém o GPS ativo quando a tela é bloqueada durante uma corrida (modo bolso).
  • BLUETOOTH_CONNECT / BLUETOOTH_SCAN — opcionais; usadas apenas se o usuário configurar um receptor GPS Bluetooth externo.
  • POST_NOTIFICATIONS — exibe a notificação de corrida em andamento durante uma sessão ativa.

Nenhuma dessas permissões é usada fora do contexto descrito acima.

Compartilhamento de dados

O Lapzy não vende, aluga nem compartilha dados pessoais com terceiros. Os dados de corrida ficam armazenados localmente no dispositivo do usuário. Nenhuma informação é transmitida a servidores externos de forma automática.

Retenção e exclusão

Todos os dados (corridas, traçados, preferências) são armazenados exclusivamente no dispositivo. O usuário pode excluir qualquer dado a qualquer momento dentro do próprio app, ou desinstalar o app para remover tudo.

Alterações nesta política

Caso esta política seja alterada de forma relevante, a data de "Última atualização" será modificada. Mudanças significativas serão comunicadas nas notas de versão do app na Google Play.
